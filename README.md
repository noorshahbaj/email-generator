## Installation

You can install the package via composer:

```bash
composer require noorshahbaj/email-generator
```

## Usage

```php
$skeleton = new Noorshahbaj\EmailGenerator();
echo $skeleton->echoPhrase('Hello, Noorshahbaj!');
```

## Testing
