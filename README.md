## Installation

You can install the package via composer:

```bash
composer require noorshahbaj/email-generator
```

## Usage

```php
use Noorshahbaj\EmailGenerator\Email;


echo Email::generate();
```

## Testing

```bash
composer test
```

or you can also try this

```bash
./vendor/bin/phpunit
```
